# 文獻回顧：院內心跳停止 (IHCA) First Responder 問題

> 整理日期：2025-12-20
> 目的：支持本專案研究背景，說明 IHCA 現有問題與介入需求

---

## 一、IHCA 整體現況

### 存活率偏低

| 來源 | 發現 |
|------|------|
| Turkington 2014 (PMID: 24257161) | IHCA 存活率僅 **17%** |
| Vestergaard 2021 (PMID: 34938129) | 30 天存活 25%，1 年 21%，5 年 13% |
| 多項研究 | 存活率雖逐年改善，但仍遠低於理想 |

### 存活關鍵因素

> 「Survival depends mainly on the effectiveness of the emergency response」
> — Resuscitation 2004 (PMID: 15325448)

---

## 二、CPR 品質問題（即使受過訓練）

### 關鍵文獻：Abella et al. 2005 (JAMA)

**Quality of cardiopulmonary resuscitation during in-hospital cardiac arrest**
(PMID: 15657323)

| CPR 品質參數 | 發現 | 指引標準 |
|-------------|------|---------|
| **壓胸頻率不足** | 28.1% 的時段 < 90/min | 100-120/min |
| **壓胸深度不足** | 37.4% 的壓胸 < 38 mm | ≥ 50 mm |
| **過度換氣** | 60.9% 的時段 > 20/min | 10-12/min |
| **No-flow fraction** | 平均 0.24 | 建議 < 0.20 |

**結論**：
> 「The quality of multiple parameters of CPR was inconsistent and **often did not meet published guideline recommendations**, even when performed by **well-trained hospital staff**」

### Vestergaard 2021 (PMID: 34938129)

| 參數 | 測量值 | 問題 |
|------|--------|------|
| 壓胸頻率 | 中位數 **126/min** | 超過上限 (120/min) |
| Chest Compression Fraction | **78%** | 低於建議 80% |

**意義**：即使有專責團隊，CPR 品質仍難達標

---

## 三、夜間/週末（Off-Hours）更糟

### 存活率差異

| 來源 | 發現 |
|------|------|
| 多項系統性回顧 | 夜間/週末 IHCA 存活率**顯著較低** |
| 瑞典研究 2023 | 日間 30 天存活 36.8%，夜間下降 |
| 27 篇相關文獻 | 一致發現 off-hours 結果較差 |

### 夜間/週末問題

| 問題 | 說明 |
|------|------|
| **人力不足** | 護理師人力減少，無法立即支援 |
| **急救團隊反應延遲** | 團隊成員分散各處 |
| **經驗較少** | 值班人員 CPR 經驗可能較少 |
| **設備取得慢** | 夜間找急救車、AED 較困難 |

---

## 四、病房 First Responder 特殊挑戰

### Singapore General Hospital 2024 (PMID: 38331344)

**Ward-based IHCA 品質改善計畫**

| 改善前 | 改善後 | 改變 |
|--------|--------|------|
| Call center 處理時間 1.8 min | 1.4 min | ↓ 22% |
| 急救團隊反應時間 5.1 min | 3.6 min | ↓ 29% |
| ROSC 率 | 改善 (OR 2.05, p=0.04) | ✓ |
| 存活出院率 | 無顯著改善 (OR 0.71) | ✗ |

**意義**：縮短反應時間可改善 ROSC，但 first responder 在等待期間的 CPR 品質仍是關鍵

### 護理師面臨的挑戰

| 挑戰 | 來源 |
|------|------|
| **焦慮與壓力** | 1,346 篇相關文獻探討護理師 CPR 時心理壓力 |
| **自信心不足** | 不常執行 CPR 導致自我效能低 |
| **流程記憶困難** | 壓力下難以回想正確順序與時間 |
| **單獨面對** | 夜班人力少，可能需獨自開始急救 |

## 五、現有問題總結

### IHCA First Responder 面臨的困境

```
┌─────────────────────────────────────────────────────────┐
│                  IHCA First Responder 困境              │
├─────────────────────────────────────────────────────────┤
│  1. 技術層面                                            │
│     • 壓胸頻率/深度難以維持標準                         │
│     • 過度換氣常見                                      │
│     • Hands-off time 過長                               │
│                                                         │
│  2. 流程層面                                            │
│     • 2 分鐘循環計時困難                                │
│     • Epinephrine 給藥時間易遺忘                        │
│     • 角色分配混亂                                      │
│                                                         │
│  3. 心理層面                                            │
│     • 壓力大、焦慮高                                    │
│     • 自信心不足                                        │
│     • 經驗少導致恐懼                                    │
│                                                         │
│  4. 環境層面                                            │
│     • 夜間/週末人力不足                                 │
│     • 急救團隊反應延遲 (平均 3-5 分鐘)                  │
│     • 等待期間品質難維持                                │
└─────────────────────────────────────────────────────────┘
```

## 六、現有解決方案的限制

| 現有方案 | 限制 |
|---------|------|
| **急救團隊 (Code Team)** | 反應時間 3-5 分鐘，first responder 仍需獨撐 |
| **AED 內建語音** | 僅限電擊流程，無完整 ACLS 時間提示 |
| **Real-time feedback device** | 設備昂貴、需培訓、不一定隨時可得 |
| **定期訓練** | 技能衰退快，實際執行時仍易遺忘 |

## 七、Research Gap：為何需要語音引導系統

### 現有證據缺口

| 缺口 | 說明 |
|------|------|
| **First responder 支援不足** | 多數研究聚焦急救團隊，非第一反應者 |
| **簡易介入研究稀少** | 多數研究探討複雜設備或 DA-CPR |
| **院內情境證據不足** | 語音引導研究多為 OHCA |
| **非急救專責人員** | 缺乏針對病房護理師的研究 |

### 本專案填補的缺口

> 「在急救團隊抵達前的 3-5 分鐘，first responder 需獨自維持高品質 CPR。現有研究顯示即使受過訓練，CPR 品質仍常不達標。本專案探討**低成本、易部署的語音引導系統**能否協助 first responder 在等待期間維持 CPR 品質。」
