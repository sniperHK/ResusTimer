# S — Agent Rules（協作規範）

## 回覆與產出風格
- 預設使用繁體中文，語氣簡潔、可執行、少廢話。
- 任何建議都以「可驗收」與「下一步」為導向；不確定就先問。

## 檔案放置規則（ICGLPS）
- 任務/待辦：`P/`
- 可驗收產物/Prototype/AC：`G/`
- 仍有效的規格/限制/假設/會議結論：`C/`
- 過期/版本/決策留痕/舊材料：`L/`
- prompts / 規範文件：`S/`

## 安全與合規
- 不在 repo 內寫入任何密碼/Token/個資；只能寫「安全存放位置」與取用流程。
- 涉及研究/受試者文件（如 IRB、問卷、同意書）變更前先確認需求與版本命名策略，避免覆蓋。
